# Getting Started

Welcome to **Chasing Cloud Careers**! This is your starting point for free, self-service training materials covering cloud computing, AI, DevOps, and more.

## What You'll Find Here

Our learning paths are designed to take you from beginner to proficient through hands-on, practical content:

- **Cloud Computing** — AWS, Azure, GCP fundamentals and beyond
- **AI & Machine Learning** — From concepts to implementation
- **DevOps & Infrastructure** — CI/CD, containers, infrastructure as code

## How to Use These Paths

1. **Start here** — Read through this introduction
2. **Pick a path** — Choose the topic that interests you most
3. **Follow along** — Each path is structured with step-by-step guides
4. **Practice** — Apply what you learn with hands-on exercises
